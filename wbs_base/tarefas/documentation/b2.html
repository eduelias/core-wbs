What is b2 and why would I want to post to it?

b2 is a nice PHP/MySQL blog/web journal package. I generally keep ideas for new entries and partially written entries in Tasks Jr. and wanted to be able to post new blog entries directly from tasks.

If you have it enabled, there is a small 'b2' button in the toolbar on the task Edit screen. Press that button and it will send the task to the b2tasks_post.php page. From there, you can make any edits you want and post it to your blog.
