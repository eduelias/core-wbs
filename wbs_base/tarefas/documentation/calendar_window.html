Why does the Calendar open in a new window?

PHP iCalendar, the software used to display the Calendar, is released under a GPL license. Part of that license says that if your software relies on the GPL'ed software then you must also release under GPL. I don't want to release under GPL so I open PHP iCalendar in a new window.
