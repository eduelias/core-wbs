Why doesn't Tasks Jr. have categories?

I've had several people ask me to add categories to Tasks Jr. I haven't done so because we already have the tree hierarchy for organization and I think an additional organization method would add another choice that would make Tasks Jr. more complex and difficult to use without bringing much benefit.
