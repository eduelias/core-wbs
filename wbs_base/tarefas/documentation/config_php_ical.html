Configuring PHP iCalendar

If enabled, the Calendar button on the toolbar will open the included PHP iCalendar in a new browser window, populated with your tasks data for the range you specified in config.php.

If this is not working, make sure that:

  • The paths to PHP iCalendar are correct in your Options and Settings (config.php).
  • You do not have pop-up blocking software that is stopping the window from opening. If the window will not open for you on the demo then there is a configuration problem with your browser.
  • If you are have added 401 authentication (using .htaccess to prompt for a username/password) to your Tasks Jr. installation and are having trouble opening the tasks iCalendar in PHP iCalendar, you may need to add the URL of the tasks iCalendar to the PHP iCalendar config.inc.php file.
