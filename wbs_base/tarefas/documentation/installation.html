Installing Tasks Jr.

When you have PHP and MySQL installed and configured, you're ready to install Tasks Jr.

You should already know how to add files and directories to you web server, if you do not you're going to need to get a quick lesson from your local web geek before you can proceed.

Unzip the Tasks Jr. zip file that you downloaded (keeping the folder hierarchy intact) and put the files into a directory in your web server.
