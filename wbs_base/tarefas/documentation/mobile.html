How do I use the mobile/PDA version?

Most mobile users can just hit the main page because we try to detect mobile browsers are redirect them. If your mobile browser isn't detected, you can type in the address to the mobile.php page (http://www.example.com/tasks/mobile.php). Please point your mobile browser to this page so we can add it to our list.
