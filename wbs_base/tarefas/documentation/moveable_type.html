What is Moveable Type and why would I want to post to it?

Moveable Type is a blog/web journal package written in PERL. You can keep ideas for new entries and partially written entries in Tasks Jr. before you post them on your blog. You can then post new blog entries directly from Tasks Jr.

If you have it enabled, there is a small 'MT' button in the toolbar on the task Edit screen. Press that button and it will send the task to the Moveable Type bookmarklet. From there, you can make any edits you want and post it to your blog.
