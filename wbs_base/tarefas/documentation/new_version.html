How do I know if there is a new version of Tasks Jr.?

There are several ways to stay up to date.

  1. Assuming you have it enabled in config.php, going to the about page will show you the version of Tasks Jr. you are running and any new releases (with download links).
  2. The Tasks dev-blog has the latest news on my tasks development.
  3. The Tasks Jr. web page will also have the latest version available (of course).
