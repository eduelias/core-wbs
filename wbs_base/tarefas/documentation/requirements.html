Tasks

Requirements

Tasks Jr. requires PHP and MySQL. PHP and MySQL are free downloads and will run happily on Mac OS X, Windows NT/2000/XP and Unix/Linux systems.

PHP Installation

PHP is required to run Tasks Jr., here are some web sites that are good resources for getting PHP up and running. PHP must be configured to allow short PHP tags (this is the default setting).

  • PHP Installation on Mac OS X (recommended)
  • PHP Installer for Apache 2 on Mac OS X
  • PHP Installation on Windows
  • The PHP web site

MySQL Installation

MySQL is required to run tasks, here are some web sites that are good resources for getting MySQL up and running.

  • MySQL Installation on Mac OS X (recommended)
  • MySQL Installer for Mac OS X
  • MySQL Installation on Windows
  • The MySQL web site
