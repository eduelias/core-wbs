Tasks

Home Screen

This screen shows you a Tree View of all of your root tasks (tasks with no parent tasks)

All open (not completed) root tasks are shown in the top list titled "Open Root Tasks". If a root task is "pinned" (addlink: see more about pinned tasks here), it will remain on the Home screen even after it is completed. These completed tasks will be a list below the "Open Root Tasks" list called "Completed Root Tasks".
