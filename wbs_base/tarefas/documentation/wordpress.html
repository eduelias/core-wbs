What is WordPress and why would I want to post to it?

WordPress is a nice PHP/MySQL blog/web journal package. It is the successor to b2, which I use to run my blog. I generally keep ideas for new entries and partially written entries in Tasks Jr. and wanted to be able to post new blog entries directly from Tasks Jr.

If you have it enabled, there is a small 'WP' button in the toolbar on the task Edit screen. Press that button and it will send the task to the WordPress bookmarklet. From there, you can make any edits you want and post it to your blog.
